# ai-based-farmer-advisory-system

## Features
- AI powered farmer query assistance
- Answers related to crops,soil,fertilizers,and plant diseases
- Uses Large Language Models for intelligent responses

## Tech Stack
- Frontend : HTML,CSS,JavaScript
- Backend : FASTAPI (Python)
- AI Model : Llama 3.2 (Via Ollama)
- Development Tools : VS Code,GitHub
- Platform : Web Application

- Course : B.Tech Computer Science
- Domain : Artificial Intelligence & Agriculture
